# pvz-qwh —— 主程序 + 无尽模式

本项目在原版《植物大战僵尸》(PlantsVsZombies) 的基础上，把另一份独立的
**程序化绘制无尽模式** 作为可选模式缝入开始界面。两者各自独立、互不干扰。

## 三种模式（开始界面可选）

| 按钮 | 说明 | 来源 |
|------|------|------|
| **冒险模式 (Adventure)** | 原版关卡制 PvZ，含卡牌选择、保龄球关卡、通关流程 | 原项目（未改动） |
| **多人模式 (Multiplayer)** | 原版联机对战（植物 vs 僵尸玩家） | 原项目（未改动） |
| **无尽模式 (Endless)** | 程序化绘制、无尽波次 + 天赋系统 + 特殊子弹 | 新增子游戏 |

## 运行

```bash
pip install -r requirements.txt   # pygame>=1.9, websockets>=10.0
python main.py
```

在开始界面点击「无尽模式」即可进入无尽子游戏。

## 无尽模式操作

- 鼠标点击选择植物 / 种植；数字键 1–0、Q W E R T 快捷选植物
- 空格收集阳光；X 铲子模式；ESC 取消选择
- F1/F2/F3 或右上角按钮切换 1x/2x/3x 倍速
- **按 B 返回主菜单**（或在该子游戏主菜单按 ESC）

### 无尽模式特色

- **无尽波次**：僵尸随波次增强，难度递增
- **天赋池系统**：每过一波从 6 个天赋中随机抽 3 个选 1，跨波累积
  - 豌豆精通 / 光合作用 / 投掷强化（可叠加 +10%）
  - 豌豆百变（70% 豌豆 / 10% 黄油 / 10% 星星秒杀 / 10% 爆炸豌豆 3×3）
  - 双倍阳光（向日葵 50% 概率双产）
  - 连环投掷（投手类 50% 概率连击两次）
- **特殊子弹**：黄油(冻结+高伤)、星星(秒杀)、爆炸豌豆(范围伤害)
- **火炬树桩**：普通豌豆→火豌豆(伤害×2)；寒冰豌豆→普通豌豆

## 代码结构（新增部分）

```
source/
├── state/endless.py      # 新增：无尽模式包装状态（resize + 调用子游戏）
├── endless/              # 新增：独立无尽子游戏
│   ├── constants.py      #   常量 / 天赋表 / 植物&僵尸信息
│   ├── sprites.py        #   程序化绘制所有精灵
│   ├── game.py           #   游戏逻辑(可独立运行，也可嵌入)
│   └── assets/           #   植物立绘等图片(含火炬树桩/玉米/毁灭菇/西瓜)
```

宿主侧的改动（均为最小侵入）：
- `source/constants.py`：新增 `ENDLESS` 状态名
- `source/main.py`：注册 `Endless` 状态
- `source/state/mainmenu.py`：开始界面新增「无尽模式」按钮
- `source/state/endless.py`：进入即把窗口切到 1200×700，跑完子游戏再切回 800×600
- `source/tool.py`：每帧重新获取屏幕表面（兼容尺寸切换）；支持子游戏请求关闭整个窗口

> 原版冒险模式与多人模式的逻辑、贴图、关卡数据均**未改动**。
